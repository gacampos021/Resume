Bitcoin x Ethereum - Data Analysis Project

This project aimed to demonstrate skills in data pre-processing (cleaning) and data analysis. To achieve this objective, the Python language and libraries such as Pandas, Matplotlib, and Seaborn were utilized on a Kaggle dataset containing Bitcoin and Ethereum prices up to 2025.

The data comprised eight CSV files, which were consolidated into two distinct datasets: one for Bitcoin and one for Ethereum. The process involved reading each relevant file and conducting thorough null value checks. Rows containing null values were removed. Column names were standardized to ensure uniformity across datasets, and columns unique to specific files were dropped. Finally, data types were transformed to their appropriate formats, with date columns converted to datetime objects and numerical columns to integer or float types. Upon completion of these transformations, the datasets were successfully integrated.

Following data integration, visualizations were generated to analyze the datasets. Initially, time-series plots were created to visualize the price trends over time. Subsequently, annual and monthly percentage change rates were calculated and presented using bar graphs to illustrate the growth patterns.

Finally, a comparative analysis was conducted to demonstrate the direct influence of Bitcoin prices on Ethereum prices, visualized through a relative price graph.
